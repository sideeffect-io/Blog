Dépannage

  1. Mettez à jour Funny Boum vers la dernière version.
  2. Forcez la fermeture puis rouvrez l’app.
  3. Redémarrez votre iPhone.
  4. Réinstallez l’app si le problème persiste.

Questions fréquentes

Comment signaler un bug ?

Envoyez un email à thibault@sideeffect.io avec une description du problème et les étapes pour le reproduire.

Funny Boum collecte-t-il des données personnelles ?

Non. Funny Boum ne collecte, ne stocke ni ne transmet aucune donnée personnelle. Le jeu fonctionne entièrement sur votre appareil.

Le jeu plante — que faire ?

Suivez les étapes de dépannage ci-dessus. Si le problème persiste, contactez le support avec le modèle de votre appareil et la version d’iOS.
